# API Reference Overview

This site uses the standard Mintlify split:

- `Guides` for workflows, topology, and conceptual API overviews
- `API Reference` for endpoint docs generated directly from OpenAPI

## API surfaces

- `GMGN Token`
- `GMGN Wallet`
- `OKX Trending`
- `Padre Trending`

## How to read this section

1. Start with the overview page for the API surface you need.
2. Open the autogenerated endpoint group in the sidebar.
3. Use `OpenAPI Files` if you want the raw spec artifacts for external tools.

## Source of truth

The endpoint reference pages are generated from:

- `openapi/gmgn-token.openapi.json`
- `openapi/gmgn-wallet.openapi.json`
- `openapi/okx-trending.openapi.json`
- `openapi/padre-trending.openapi.json`

If a prose page and an endpoint page disagree, treat the OpenAPI-backed endpoint page as the source of truth.
