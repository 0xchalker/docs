# Quickstart

## 1. Start the local API server

```bash
npm run gmgn:api
```

This serves:

- `GET /health`
- `GET /openapi.json`
- `GET /docs`
- `GET /api/gmgn/token`
- `GET /api/gmgn/token/*`
- `GET /api/gmgn/wallet`
- `GET /api/gmgn/wallet/*`

## 2. Start the terminal

```bash
npm run gmgn:ui
```

Open:

- `http://localhost:3001/`
- `http://localhost:3001/sol/token/{mint}`
- `http://localhost:3001/sol/address/{wallet}`

## 3. Start Mintlify docs

```bash
npm run docs:openapi:dev
```

Open:

- `http://localhost:3000/`

Note:

- `localhost:3000` is reserved for Mintlify docs only
- the generic `npm run dev` command now uses `localhost:3004` to avoid colliding with docs

## 4. Regenerate OpenAPI files

```bash
npm run docs:openapi:generate
```

## 5. Validate the generated specs

```bash
npm run docs:openapi:check
```

## 6. Read the docs

- `Guides` explains the local stack and page behavior
- `API Reference` is generated from OpenAPI files and is the source of truth for endpoint contracts
