# Market Crawler API Suite

This docs site documents the local crawler stack that runs in this workspace:

- `Mintlify docs` on `http://localhost:3000`
- `GMGN UI terminal` on `http://localhost:3001`
- `GMGN API` on `http://127.0.0.1:3002`

## Site structure

- `Guides`
  - quickstart, services, terminal behavior, and data-model pages
- `API Reference`
  - overview pages for each API surface
  - endpoint pages generated from OpenAPI

## In scope

- GMGN token and wallet APIs
- OKX trending snapshot API
- Padre snapshot and streaming APIs
- local runtime OpenAPI and Swagger output

## Out of scope

- fabricated private GMGN data
- login-gated tracker features exposed as public local APIs
- hosted SaaS deployment behavior that does not exist in this workspace
