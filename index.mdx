# GMGN Local Terminal

This docs site now tracks the local GMGN stack that runs in this workspace:

- `Mintlify docs` on `http://localhost:3000`
- `GMGN UI terminal` on `http://localhost:3001`
- `GMGN API` on `http://127.0.0.1:3002`

## What is in scope

- token detail terminal
- wallet detail terminal
- local Swagger and OpenAPI output
- section-based token endpoints for table tabs
- browser-session-backed wallet endpoints split by wallet page section

## What is intentionally not in scope

- fake login-gated GMGN tracker data
- private monitor streams fabricated for local UI
- share and tracker rails as public local wallet features
- generic public SaaS hosting
