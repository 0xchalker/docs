# OpenAPI Files

These files are generated into the repo and can be imported into Swagger UI, Postman, Insomnia, Stoplight, or other tooling.

## Files

- `openapi/gmgn-token.openapi.json`
- `openapi/gmgn-wallet.openapi.json`
- `openapi/okx-trending.openapi.json`
- `openapi/padre-trending.openapi.json`

## Generation

These specs are generated from the crawler app repo and committed here for Mintlify publishing.

## Validation

Validate them with:

```bash
npx mint openapi-check openapi/gmgn-token.openapi.json
```

## Notes

- `GMGN` also exposes a runtime Swagger page at `http://127.0.0.1:3002/docs`
- `GMGN` runtime OpenAPI JSON is available at `http://127.0.0.1:3002/openapi.json`
- the generated specs assume local default base URLs unless you override them before generation
- the wallet spec documents `profile`, `stats`, `holdings`, `activity`, and `rankings` as first-class endpoints
